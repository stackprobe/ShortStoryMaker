# Plot-03

- 仮題：湾の言葉
- 舞台となる日付と場所：1861年5月21日前後、日本、対馬、浅茅湾周辺
- 主人公：対馬藩の通訳見習い佐市。20代前半。外国語を断片的に覚えただけの架空人物。
- 主人公の目的：ロシア水兵と村人・藩士の衝突を避け、漁師である兄を無事に海へ戻す。
- 障害：ロシア側の意図が読めないこと、藩士の面子、村人の恐怖、通訳としての未熟さ、中央の幕府方針が現地に十分伝わらないこと。
- 使用する史実上の出来事：ポサドニック号の対馬滞在、5月の島民・武士とロシア水兵の衝突、幕府の退去交渉。
- 使用する物語素材：浅茅湾の軍艦、外国語の号令、測量図、警備縄、通訳の覚書、漁網、藩からの触書、潮と煙の匂い。
- 結末の方向性：佐市は自分の訳が不確かだと公の場で認め、面子を失う。衝突そのものは完全には止められないが、報復の拡大を防ぐため、双方が誤解していた一点を明らかにする。
- この案の魅力：外交事件を、中央の政治ではなく島の生活者の恐怖と言葉の不足から描ける。主人公の勝利が小さく、苦い余韻を残せる。
- 史実との距離感：ポサドニック号滞在と5月の衝突、幕府対応は史実に沿う。主人公、兄、具体的な訳の失敗は創作上の補完。重大人物を英雄化・悪役化しない。
