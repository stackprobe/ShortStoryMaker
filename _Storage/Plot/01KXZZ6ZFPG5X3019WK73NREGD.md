# Plot-01

- 仮題：泥道の皿
- 舞台となる日付と場所：1861年5月下旬、ニュージーランド、ダニーデンからガブリエルズ・ガリーへ向かう道
- 主人公：ダニーデンの道具商エレン。30代。夫を亡くし、小さな店と弟を抱えている架空人物。
- 主人公の目的：金鉱へ向かう人々に採掘皿とシャベルを売り、借金で失いかけた店を守る。
- 障害：在庫不足、泥道と寒さ、未払いの客、弟が自分も金を掘ると言い出すこと、金発見の噂の真偽が分からないこと。
- 使用する史実上の出来事：1861年5月下旬のガブリエルズ・ガリーでの金発見と、オタゴ・ゴールドラッシュの始まり。
- 使用する物語素材：新聞記事、錫の皿、泥道、冷たい雨、港に着く船、報奨金の噂、留守を守る家族、金を量る秤。
- 結末の方向性：エレンは金を掘りに行くのではなく、道中で倒れた若者に最後の皿を譲る。店を救える保証は失うが、弟を連れ戻し、金に群がる側ではなく生活を支える側として新しい商売を始める。
- この案の魅力：大きな金鉱ブームを、掘る人ではなく周辺で生き残ろうとする商人の視点から描ける。希望と商売、家族、倫理の衝突が短編向きにまとまる。
- 史実との距離感：ガブリエル・リードの発見とゴールドラッシュは史実に沿う。主人公、弟、個別の道中事件は創作上の補完。
