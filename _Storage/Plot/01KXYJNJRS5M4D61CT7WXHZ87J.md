# Plot-01

- 仮題: 折れた鉛筆
- 舞台となる日付と場所: 1955年9月23日、アメリカ合衆国ミシシッピ州サムナー、タラハチー郡裁判所周辺
- 主人公: 裁判所の黒人清掃員。40代。開廷中は法廷の外に追いやられるが、廊下や清掃中に証言の断片を聞く。
- 主人公の目的: 評決の日、証人の親族が無事に町を出られるよう裏口まで案内し、同時に自分が見聞きしたことを忘れずに持ち帰る。
- 障害: 裁判所内の人種隔離、白人住民の視線、職を失う恐怖、何を語れば誰が危険になるか分からない状況。
- 使用する史実上の出来事: エメット・ティル殺害事件の公判と、1955年9月23日の無罪評決。
- 使用する物語素材: `Material/emmett-till-trial-1955.md`の清掃員、隔離された傍聴席、評決後の沈黙、折れた鉛筆、記録することと生存の対立。
- 結末の方向性: 主人公は証人の親族を町外れまで送る。法廷の床で拾った折れた鉛筆を捨てられず、名前を出さない形で、聞いた言葉を家族にだけ語る。
- この案の魅力: 大事件の中心人物を主人公にせず、床を磨く人物の視点から、法廷の空気、沈黙、記録されない記憶を描ける。
- 史実との距離感: 公判日程と無罪評決は史実に沿う。清掃員、拾われる鉛筆、証人親族の移動は創作上の補完。後年の再捜査情報は登場人物の認識に入れない。
