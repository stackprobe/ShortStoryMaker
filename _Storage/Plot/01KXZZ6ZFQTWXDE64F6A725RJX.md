# Plot-02

- 仮題：色のない妹
- 舞台となる日付と場所：1861年5月17日、イギリス、ロンドンの王立研究所と写真師の作業場
- 主人公：写真師の助手トム。10代後半。薬品の扱いは上手いが、正式な職人としては認められていない架空人物。
- 主人公の目的：亡くなった妹の白黒写真に、記憶の中の髪と服の色を残す手がかりを得る。
- 障害：三色分解の実演は理論の可能性を示すだけで実用には遠いこと、主人の命令、講演会場での身分差、家族が「本当の色」を求めていること。
- 使用する史実上の出来事：ジェームズ・クラーク・マクスウェルによる三色分解カラー写真の実演。
- 使用する物語素材：タータンリボン、赤・緑・青のフィルター、ガラススライド、講演会の入場券、薬品の匂い、手彩色の筆、暗い講堂。
- 結末の方向性：トムは妹の写真を「完全なカラー写真」として偽ることをやめ、不完全な三色の試し刷りと、手で色を添えた白黒写真の両方を母に渡す。
- この案の魅力：技術史を説明せず、未完成の技術が個人の喪失と記憶に触れる瞬間として描ける。成功と失敗の曖昧さが余韻になる。
- 史実との距離感：実演、日付、場所、タータンリボンの使用は史実に沿う。助手トムと妹の写真は創作上の補完。実用カラー写真として描きすぎない。
